# VibeTalent

## The Marketplace Where Quality and Consistency Beat Resumes

VibeTalent is an AI-powered talent marketplace that ranks developers ("vibecoders") based on **proof of work** — shipped projects, daily coding streaks, and a composite reputation score called the **Vibe Score**. Instead of resumes and interviews, clients hire vibecoders based on an unfakeable signal: real output.

### The Core Idea

> The best signal of a great developer is what they've actually built and shipped.

Traditional hiring is broken for indie developers and vibecoders. VibeTalent solves this by tracking what actually matters:

- **Projects** — What have you actually built and deployed? Quality projects with live demos are the strongest signal.
- **Streaks** — How many consecutive days have you shipped code?
- **Vibe Score** — A composite score that combines project quality, streaks, and badge level into a single hiring signal.

### How It Works

1. **VibeCoders sign up** via GitHub or Google OAuth
2. **Ship projects** with live URLs and GitHub repos as proof
3. **Log daily activity** to build streaks and earn badges
4. **Get discovered** on the public leaderboard and explore page
5. **Get hired** — clients send hire requests directly through the platform

### Key Features

| Feature | Description |
|---|---|
| **Vibe Score** | Composite reputation score based on project quality + streaks + badges |
| **Project Showcase** | Verified projects with live demos, screenshots, and tech stack details |
| **Streak System** | Daily activity tracking with automatic badge progression |
| **Badge Levels** | Bronze (30d), Silver (90d), Gold (180d), Diamond (365d) |
| **Public Profiles** | Activity heatmaps, project showcases, and reviews |
| **Hire Requests** | Direct messaging between clients and vibecoders |
| **AI Agent** | Intelligent vibecoder matching and evaluation |
| **Public API** | OpenAPI 3.0 spec for integrations |
| **GitHub Sync** | Auto-import coding activity from GitHub |

### Who Is This For?

- **VibeCoders** who want to showcase their projects, prove their consistency, and get discovered
- **Clients** looking to hire reliable developers based on real track records
- **AI Agents** that need structured talent data via the public API
