# VibeTalent

## The Marketplace Where Consistency Beats Resumes

VibeTalent is an AI-powered talent marketplace that ranks developers ("vibecoders") based on **proof of work** — daily coding streaks, shipped projects, and a composite reputation score called the **Vibe Score**. Instead of resumes and interviews, clients hire builders based on an unfakeable signal: consistency.

### The Core Idea

> A 90-day coding streak proves more about a developer than any portfolio ever could.

Traditional hiring is broken for indie developers and vibecoders. VibeTalent solves this by tracking what actually matters:

- **Streaks** — How many consecutive days have you shipped code?
- **Projects** — What have you actually built and deployed?
- **Vibe Score** — A composite score that combines streaks, projects, and badge level into a single hiring signal.

### How It Works

1. **Builders sign up** via GitHub or Google OAuth
2. **Log daily activity** to build streaks and earn badges
3. **Ship projects** with live URLs and GitHub repos as proof
4. **Get discovered** on the public leaderboard and explore page
5. **Get hired** — clients send hire requests directly through the platform

### Key Features

| Feature | Description |
|---|---|
| **Vibe Score** | Composite reputation score based on streaks + projects + badges |
| **Streak System** | Daily activity tracking with automatic badge progression |
| **Badge Levels** | Bronze (30d), Silver (90d), Gold (180d), Diamond (365d) |
| **Public Profiles** | Activity heatmaps, project showcases, and reviews |
| **Hire Requests** | Direct messaging between clients and builders |
| **AI Agent** | Intelligent builder matching and evaluation |
| **Public API** | OpenAPI 3.0 spec for integrations |
| **GitHub Sync** | Auto-import coding activity from GitHub |

### Who Is This For?

- **Vibecoders** who want to prove their consistency and get discovered
- **Clients** looking to hire reliable developers based on real track records
- **AI Agents** that need structured talent data via the public API
